### App Login - Firebase

App de Login desenvolvido na stack:

1. Angular
2. Typescript
3. Ionic Framework
4. Firebase
5. Html

#### Instalação
1. Clonar projeto via GIT para Localhost:
`git clone`

2. Executar npm install:
`npm install`
